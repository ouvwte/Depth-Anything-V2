# Тестирование Depth-Anything-V2 в Docker

В этом репозитории находится полностью автономное окружение для проверки
воспроизводимости модели Depth-Anything-V2.  
При запуске Docker-контейнера автоматически выполняются следующие шаги:

- загрузка оригинальной модели Depth-Anything-V2,
- инференс на тестовых изображения `test_image_1.jpg` и `test_image_2.jpg`,
- сравнение полученной карты глубины с эталонной (`depth_reference.npy`),
- вычисление метрик: абсолютная разность матриц, MSE, SSIM,
- вывод результатов в консоль и сохранение карты разности `difference_map.png`.

## Требования

- Установленный [Docker](https://docs.docker.com/get-docker/) (любая современная версия).
- Доступ в интернет **только на этапе сборки образа** (для загрузки базового образа PyTorch и Python-зависимостей). Сама модель и тестовые данные уже находятся внутри репозитория.
- Свободное дисковое пространство > 8 GB.

## Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/ouvwte/Depth-Anything-V2.git
cd Depth-Anything-V2
```

### 2. Соберите Docker-образ
```bash
docker build -t depth-anything-test . 
```
Внимание: образ включает веса модели(~100 Mb) и будет весить примерно 3.08 ГБ (только устанавливаемый контент). Сборка может занять несколько минут.

### 3. Запустите тестирование

Если у вас нет видеокарты NVIDIA, используйте команду: 
```bash
docker run --rm depth-anything-test
```
В этом случае программа отработает на CPU (универсальная команда).

При наличии видеокарты NVIDIA, для ускоренного инференса, используйте команду:
```bash
docker run --gpus all --rm depth-anything-test
```
